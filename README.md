
# Placement Predictor Model

This project predicts whether a student is likely to get placed based on academic and skill-based inputs.

## Features
- User input via web interface
- Machine Learning model (Logistic Regression)
- Customizable website background (solid, gradient, texture)

## Tech Stack
Python, Pandas, NumPy, Scikit-learn, Streamlit, Matplotlib

## Run Instructions
1. Install requirements:
   pip install -r requirements.txt
2. Run app:
   streamlit run app.py
